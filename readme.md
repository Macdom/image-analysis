Program służy do przeprowadzania operacji na jednym lub dwóch obrazach, takich jak:
a) zmiana jasności
b) zmiana kontrastu
c) korekcja gamma
d) rozszerzenie histogramu
e) wyrównanie histogramu
f) mnożenie dwóch obrazów
g) przeplot dwóch obrazów.
